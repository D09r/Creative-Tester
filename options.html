FAQs

How to create a custom proxy profile?

In Proxy Profiles, click New Profile button to create a new proxy profile.
Type Profile Name, Protocol Type (optional), Proxy Server, Port, BypassList (optional) and click Save proxy settings to save your custom proxy profile.

How to edit custom proxy profiles?

In Proxy Profiles, click on the proxy profile which need to be edit.

How to disable proxy or to enable direct connection mode?

Click icon on top right of Chrome window, and click Disable Proxy option to enable direct connection mode which disables proxy connection and traffic passes directly to internet.

How to enable system proxy connection mode?

Click icon on top right of Chrome window, and click System Proxy option to enable system proxy mode.

How to export proxy profiles?

In General, click Export Proxy Profiles to download backup file(Feed Proxy.json).

How to track Live HTTP Headers?

Click icon on top right of Chrome window, and click HTTP Headers option which opens a creative tester on a new tab.
